# ReleaseOps — Versioning Standard

## Objective
Unifier la nomenclature des tags pour éviter la dette opérationnelle.

## Conventions
### Tooling (non-core)
- Tag: `vX.Y.Z-toolsN`
- Mode par défaut: prerelease
- Scope: "ReleaseOps Tooling"

### GA (core milestone)
- Tag: `vX.Y.Z`
- Mode: GA
- Scope: "ReleaseOps GA"

### Hotfix
- Tag: `vX.Y.Z-hotfixN`
- Mode: GA
- Scope: "Hotfix"
